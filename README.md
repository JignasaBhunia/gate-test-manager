# GATE Test Manager VS Code Setup

This branch contains setup and configuration to facilitate development in VS Code with GitHub Copilot support.

## How to start development

1. Clone this repo and checkout the `vscode-setup` branch:

   ```bash
   git clone https://github.com/JignasaBhunia/gate-test-manager.git
   cd gate-test-manager
   git checkout vscode-setup
   ```

2. Open the project folder in VS Code.

3. Install the recommended extensions when prompted, especially GitHub Copilot.

4. Start editing React components. Copilot will provide code suggestions inline.

5. Run the app using your preferred React setup or web server.

## VS Code Extensions Recommended

- GitHub Copilot
- ES7 React/Redux/React-Native snippets
- ESLint
- Prettier

## Branch details

This branch will contain modular React components and VS Code config optimized for Copilot usage.
